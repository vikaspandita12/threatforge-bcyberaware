## About

ThreatForge is the SAR (Suspicious Activity Report) automation engine powering the BCyberAware platform. It streamlines the detection, generation, and management of security incident reports using AI.

## Features

- Automated SAR generation from threat data
- AI-driven anomaly detection and classification
- Streamlined compliance reporting workflows
- Integration with the BCyberAware intelligence dashboard
